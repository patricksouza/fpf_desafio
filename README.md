# FPF - Desafio técnico

    Sistema para gerenciar os cadastros de projetos, seu tempo de duração
e realizar uma simulação do cálculo de retorno do investimento.


## Tecnologias utilizadas

 * Nodejs
 * Postgresql
 * Reactjs
 * Knex
 * Bootstrap
